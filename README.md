#### 二，组队

​	在信息不对称的情况下，优先考虑三人的专业搭配，比如或信电的小伙伴负责编程和理工科题建模，经济金融统计负责论文和统计建模，数学计算专业的全方位建模以及帮忙论文，个人感觉这样子比较好。由于建模粗略地可以分为**建模，编程，论文**，三块，整体上是一人负责一块的，但是绝对不能走极端，每个人就单单的负责一块，这样子的组合缺乏沟通和互动。应该要在培训中磨合，结合每个人的个人特点。主要负责哪几块，辅助哪几块。

- **建模**
- **编程**
- **论文**
